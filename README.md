## Description 
Classic snake game in terminal written in c

## Setup
```
$ sudo apt-get install libncurses5-dev libncursesw5-dev
$ git clone https://github.com/warmi12/Csnake
$ sudo cmake CMakeLists.txt
$ sudo make
$ ./snake
```
